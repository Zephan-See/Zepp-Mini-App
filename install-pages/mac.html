If macOS says “SlideFlipper is damaged”, you can use either method below.

Recommended: download the `.command` helper, then double-click it after installing the DMG.

xattr -cr /Applications/SlideFlipper.app
Download .command Helper

Or copy the command above and paste it into Terminal once.

You can delete the `.command` file after it finishes running.
