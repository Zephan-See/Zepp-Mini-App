Windows Install — SlideFlipper

Turn your watch into a clicker 🖱

One command sets everything up. No coding experience needed.

1
Tap Copy below to copy the install command.
2
Open PowerShell — press Win + R, type powershell, press Enter.
3
Paste (Ctrl + V) and press Enter. Done!
PowerShell
Invoke-WebRequest -Uri "https://raw.githubusercontent.com/Zephan-See/Zepp-Mini-App/main/slide-flipper/pc-controller/server.js" -OutFile "$HOME\Desktop\slideflipper.js"; node "$HOME\Desktop\slideflipper.js"
✓ Copied! Now paste into PowerShell.

After pasting & pressing Enter:

  • The controller starts and shows your PC's IP address.
  • Enter that IP in the IP SETUP screen on your watch.
  • Open your presentation, then tap FLIPPER on the watch to control it.
  • Keep PowerShell open while presenting.

Need to run it again another day? The file is saved on your Desktop as slideflipper.js.
Just open PowerShell and type: node "$HOME\Desktop\slideflipper.js"
